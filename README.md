# Java Web Demonstration
The Java Web Engineer Project Demonstrations For Beginner



**仓库的意义与目的：**

- 秉承着"简单"和"容易"这两项原则，我将Java Web中的重要的、较难的知识模块进行拆分，写成一个个独立的、可直接运行的小项目。
- 对于每一个小项目，都能够运行起来，让效果呈现出来。
- 尝试着把一个知识点讲得"简单"，尝试让初学者觉得更"容易"。
