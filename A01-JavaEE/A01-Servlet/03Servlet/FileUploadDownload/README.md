# 文件的上传与下载

**项目功能：** 模拟上传文件和下载文件

**项目意义：**

1. Servlet 3.0及以后的版本中，上传的文件将保存在Part实例中，由request对象根据上传的表单控件的name获取对应的part对象，同时还支持多个文件上传。
2. 本个项目基于Servlet 3.0及以后的版本，使用Part实例来实现上传文件的功能。
